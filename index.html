Help & FAQ

How to Add to Home Screen

iPhone (Safari)
  1. Tap the Share button
  2. Scroll down and tap Add to Home Screen
  3. Tap Add
Android (Chrome)
  1. Tap the Menu (three dots)
  2. Tap Install App or Add to Home Screen
  3. Tap Install

Camera Not Working?

  • Check permissions: Go to Settings → Camera → Allow for your browser
  • Try another browser: Use Safari (iOS) or Chrome (Android)
  • Use manual entry: Tap "Enter URL manually" below the camera

Need a New Program?

  • Tap Use New QR Code to scan a new program
  • Or ask your clerk for the latest QR code

Leadership Agenda

  • Setup: Add agenda_* rows in your main sheet with unique IDs. Create a separate private agenda sheet containing those keys/IDs and your content. Save its CSV URL in the app via the lock (🔒) button.
  • Viewing: Tap the briefcase (👔) button to toggle between public program and leadership agenda. The toggle appears only when an agenda is configured.
  • Content: The agenda appears inline where placeholders were placed. Each section is a collapsible accordion with a lock icon. Announcements and business items render as bulleted lists; general notes render as paragraphs.
  • Transition: When leadership rotates, create a new agenda sheet and share the new URL; the old URL can be removed.
  • Note: Public users see no agenda; the layout remains exactly the same.
